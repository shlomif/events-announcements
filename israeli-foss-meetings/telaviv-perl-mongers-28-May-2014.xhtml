Tel Aviv Perl Mongers Meeting on 28 May, 2014

(The Hebrew text will be followed by an English one).

ב-28 במאי 2014 (יום רביעי) נערוך את מפגש הפרל החודשי שלנו שבו ניפגש לשמוע את הרצאתו של ינון פרק על ”עורך הטקסט וים למפתחי פרל“. אנו נפגשים ב-18:30 ומתחילים ב-19:00. כתובת: מכללת שנקר, חדר 247 (הבניין הישן ברחוב אנה פרנק) ברמת גן. ראוי לציין שקשה מעט למצוא את החדר וכדי לעשות זאת יש לעלות לקומה שנייה, ואז למצוא גרם מדרגות המוביל לדלת, לעלות בגרם המדרגות ,להיכנס בדלת, ושם ממוקמים החדרים האלו

פרטים נוספים ניתן למצוא באתר של שוחרי הפרל של תל אביב, בדף הבית של פרל בישראל.

לוח הזמנים למפגש הוא:

  • 18:30 - 19:00 : התכנסות ומפגש חברתי.

  • 19:00 - 20:00 : ינון פרק - ”עורך הטקסט וים למפתחי פרל“.

    כשאנו כותבים קוד, כתיבת הקוד עצמה היא פעולה שיכולה לקחת הרבה זמן. שימוש בעורך טקסט טוב, עם התאמה נכונה שלו, יכולה לשפר מאוד את קצב העבודה. בהרצאה זו אנו נתמקד בעורך הטקסט וים, ונראה כיצד התאמה נכונה של העורך תאפשר לנו להיות יעילים יותר בעת עבודת הקידוד. אנו נתחיל עם התקנה נקייה של וים, ונלמד איך להתאים אותו לצרכים של מפתח פרל כדי ליצור תהליך עבודה יעיל.

  • 20:00 - … : נלך לבית קפה או מסעדה.

המפגש הוא חינמי וכולם מוזמנים. נתראה שם!

English Version

The next Tel Aviv Perl Mongers meeting will be held on Wednesday, 28 May 2014, and we will meet to hear Ynon Perek’s talk about “The Vim Text Editor for Perl Developers”. We will meet in Shenkar College in Ramat Gan, Room 247 in the old building (Entrance from Anna Frank St.). Finding the room is somewhat difficult, and, in order to get to it, one should climb to the second floor, find a flight of stairs that leads to a door, climb it and enter the door and the room is there.

We will gather at 18:30 and the talks are going to start at 19:00.

One can find more details on the web-site of the Tel Aviv Perl mongers, and on the homepage of Perl in Israel.

The agenda this month includes:

  • 18:30 - 19:00 : Get-together.

  • 19:00-20:00 : Ynon Perek - “The Vim Text Editor for Perl Developers”

    Editing text takes a lot of time. By using and customising a good text editor, we can get more work done in less time. In this talk I’m going to take Vim and show how a Perl developer can get a lot more productive by using it. We’ll start with a plain out-of-the-box Vim installation and add plugins and customisations to create a perl oriented workflow.

  • 20:00- : We'll go to a café or restaurant.

Attending the meeting is free-of-charge, and everyone is welcome to attend. See you there!
